# Currency

A cog to convert currencies using ExchangeRate-API.

## /currencyconvert (Slash Command)

Convert an amount from one currency to another.<br/>

 - Usage: `/currencyconvert <amount> <from_currency> <to_currency>`
 - `amount:` (Required) The amount to convert (e.g., 100)
 - `from_currency:` (Required) The currency to convert from (e.g., USD)
 - `to_currency:` (Required) The currency to convert to (e.g., EUR, peso)

## [p]currencyset

Setup currency cog.<br/>

 - Usage: `[p]currencyset`
 - Restricted to: `BOT_OWNER`

### [p]currencyset setup

Explains how to get an ExchangeRate-API key.<br/>

 - Usage: `[p]currencyset setup`
