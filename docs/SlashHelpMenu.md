# SlashHelpMenu

A simple help menu that shows all available slash commands grouped by their cog names.<br/><br/>This is a slash command version of the default help command. It will not show context menu commands.

## /help (Slash Command)

Shows all available slash commands.<br/>

 - Usage: `/help`

## [p]slashhelpset

Settings for the slash help menu.<br/>

 - Usage: `[p]slashhelpset`
 - Restricted to: `BOT_OWNER`

### [p]slashhelpset toggle

Whether the help menu should be sent as an ephemeral message.<br/>

 - Usage: `[p]slashhelpset toggle <toggle>`
