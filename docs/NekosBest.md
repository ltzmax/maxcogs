# NekosBest

Sends random images from nekos.best + RolePlay Commands.

## [p]nekoset

Settings for nekosbest cog.<br/>

 - Usage: `[p]nekoset`
 - Restricted to: `ADMIN`
 - Checks: `guild_only`

### [p]nekoset reset

Reset a user's command counts.<br/>

 - Usage: `[p]nekoset reset <member>`

### [p]nekoset resetall

Reset all command counts.<br/>

 - Usage: `[p]nekoset resetall`
 - Restricted to: `BOT_OWNER`
