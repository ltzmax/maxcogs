## Informations, install and cogs.
`[p]` is your prefix.

To add this repo:

```yaml
[p]repo add maxcogs https://github.com/ltzmax/maxcogs
```

To install any cog you want:

```yaml
[p]cog install maxcogs <cog name you want>
```
